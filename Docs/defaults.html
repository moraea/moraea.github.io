# Defaults Commands

Here, we have a list of the defaults commands that may be useful for a user running the non-Metal patches. These commands my prove to be useful from both a visuality and functionality standpoint.

# Visual

  • Change Menu Bar Text Color: defaults write -g Moraea_DarkMenuBar -bool true

  • Custom Menu Bar Colors (RGBA, less than 1): sudo defaults write /Library/Preferences/.GlobalPreferences.plist Moraea_MenuBarOverride 'R,G,B,A'

  • Simulate Metal Window Borders: defaults write -g Moraea_RimBeta -bool true

  • Beta Blurs: defaults write -g Moraea_BlurBeta -bool true

  • Selectively Re-Enable Blurs with Reduce Transparency Enabled (reboot required): defaults write (bundle identifier) Moraea_EnableTransparency 1

# Functional

  • Fix frozen Safari HTML Canvas (unneeded with Mojave QuartzCore): defaults write -g InternalDebugUseGPUProcessForCanvasRenderingEnabled -bool false

  • Fix Page Swiping on trackpads (unneeded with latest patches): defaults write -g AppleEnableMouseSwipeNavigateWithScrolls -bool true

  • Enable Beta Keyboard Backlight Fix (Disabled by default): sudo defaults write /Library/Preferences/.GlobalPreferences.plist Moraea_BacklightHack -bool true
